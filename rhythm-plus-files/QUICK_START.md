## 🎯 Setup Instructions (Choose One)

### **Tampermonkey (Recommended for Your Situation)**

**If you're on a Chromebook or low-end PC, this is your best option!**

1. **Install Tampermonkey** from your browser's app store:
   - Chrome: [Link](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobp55f)
   - Firefox: [Link](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
   - Safari: [Link](https://apps.apple.com/us/app/tampermonkey/id1482490089)

2. **Copy the Tampermonkey script:**
   - Open: `rhythm-plus-v2-tampermonkey.js`
   - Select all (Ctrl+A) and copy (Ctrl+C)

3. **Create new Tampermonkey script:**
   - Click Tampermonkey icon → "Create a new script"
   - Clear the template and paste our code
   - Save (Ctrl+S)

4. **Done!** Visit Rhythm+ v2, the GUI loads automatically

📖 **Full guide:** See `TAMPERMONKEY_SETUP.md`

---

### **Bookmarklet (Alternative)**

1. **Copy bookmarklet code:**
   - Open: `rhythm-plus-v2-bookmarklet-v2.js`
   - Remove the first line comment: `// Rhythm+ v2...`
   - Remove `javascript:` prefix if present
   - Copy everything else

2. **Create bookmark:**
   - Press Ctrl+D (or Cmd+D on Mac)
   - Edit the bookmark URL

## 🔧 Video Removal Not Working?

**The mod tries 4 different removal strategies:**

✓ Strategy 1: Direct video element hiding  
✓ Strategy 2: CSS-based removal  
✓ Strategy 3: Continuous monitoring (runs every 0.5 seconds)  
✓ Strategy 4: Background playback blocking  

**If still not working:**
1. Try clicking the button again
2. Click **ULTRA MODE** to apply all at once
3. Try opening/closing skins menu
4. Hard refresh (Ctrl+Shift+R)
5. Check browser console (F12) for errors

## 🔓 Skin Unlocking

**The mod uses 4 unlock methods:**

1. localStorage - Browser persistent storage
2. sessionStorage - Current session
3. window object - Global variables
4. DOM manipulation - HTML element modification

**If skins aren't unlocking:**
1. Click "Unlock All Skins" button
2. Skins should be immediately visible
3. If not, try opening the skins menu first
4. Click the unlock button again
5. Some games might need a page refresh
