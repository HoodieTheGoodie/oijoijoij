## 📋 Installation Steps

### Step 1: Install Tampermonkey
Tampermonkey is a userscript manager that automatically runs scripts on websites.

**Choose your browser:**
- **Chrome/Chromium**: [Install from Chrome Web Store](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobp55f)
- **Firefox**: [Install from AMO](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
- **Safari**: [Install from App Store](https://apps.apple.com/us/app/tampermonkey/id1482490089)
- **Edge**: [Install from Edge Store](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmapfignpabbj94f)

### Step 2: Copy the Script
1. Open file: `rhythm-plus-v2-tampermonkey.js`
2. **Copy ALL the code** (Ctrl+A then Ctrl+C)

### Step 3: Create New Tampermonkey Script
1. Click **Tampermonkey icon** in your browser toolbar
2. Select **"Create a new script"**
3. Replace the default template with our script
4. Press **Ctrl+S** to save

### Step 4: Test It!
1. Go to **Rhythm+ v2** website
2. The mod GUI should appear automatically in the **top-right corner**
3. If it doesn't, **refresh the page** (F5)

## ❓ Troubleshooting

### GUI Not Appearing
**Problem:** The mod doesn't show up when you visit Rhythm+ v2

**Solutions:**
1. Check that Tampermonkey is installed and enabled
2. Verify the script is in Tampermonkey (Dashboard → Scripts)
3. Refresh the page (F5)
4. Check the console (F12) for errors
5. Make sure you visited the correct Rhythm+ v2 website

### Video Still Showing
**Problem:** Clicking "Remove Background Video" doesn't hide it

**Solutions:**
1. Click the button again - continuous monitoring activates
2. Try "ULTRA MODE" to apply all optimizations
3. Check if the video element has unusual classes/ids
4. Try manually adding a CSS rule in DevTools (F12)

### Skins Not Unlocking
**Problem:** Clicking "Unlock All Skins" doesn't work

**Solutions:**
1. Make sure you're in the skins menu
2. Try clicking the button again (multiple unlock methods)
3. Check browser console for errors (F12)
4. Some games might cache - try a hard refresh (Ctrl+Shift+R)
5. Check if the game has anti-cheat (some prevent script modifications)

### Color Changes Not Showing
**Problem:** Skin colors aren't changing

**Solutions:**
1. Make sure skins are visible in the DOM first
2. Open skins menu, then apply colors
3. Try selecting specific skins instead of applying to all
4. If skins have inline styles, they might override custom colors
