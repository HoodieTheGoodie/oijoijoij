# Multi-Game Mods

This folder is the new home for per-game mod scripts.

## Structure

- `zombs-io/` active userscript work
- `yorg-io/` starter userscript and notes

## Usage

1. Open a userscript file in one of the game folders.
2. Copy it into Tampermonkey as a new script.
3. Enable and refresh the game page.

## Scope

These scripts are focused on quality-of-life UI changes, accessibility options, and convenience tooling.
